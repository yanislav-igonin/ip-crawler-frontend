# ip-crawler-frontend

Web view for all live discovered IP addresses on the internet.

[Backend](https://github.com/yanislav-igonin/ip-crawler) repo.

## How to run
1. Add `.env` file from `.env.example` and fill it with your data.
2. `npm i`
3. `npx prisma generate`
4. `npm run dev`

## Tech Stack
- TypeScript
- Next.js
- Prisma
- Tailwind CSS
